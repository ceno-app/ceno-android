Ceno (Censorship.No!) es un navegador web móvil descentralizado. Usa tecnología punto a punto para enviar páginas web a tu dispositivo móvil y cachea contenido popular con pares cooperativos. Ceno puede ser usado para evadir la censura de Internet y ayudar a otra@s a descargar páginas bloqueadas.

¡Dile NO a la censura! Instala el Navegador Ceno hoy para estar preparad@ para la próxima vez que te 🔌 desconecten. 

🚫🌴<strong> Sin conexión.</strong>
Ceno está diseñado con escenarios de apagones de internet en mente. Las páginas web son compartidos por una red global de pares, y almacenados en una caché distribuida para su disponibilidad cuando las redes tradicionales están bloqueadas o caídas.

🔓👀<strong> Desbloquea la web.</strong>
Accede cualquier página web. El contenido solicitado frecuentemente es mantenido en la red y no puede ser removido a la fuerza.

💲🌐<strong> Consume menos datos.</strong>
Al enrutar el tráfico de usuario a través de redes punto a punto, el Navegador Ceno consume menos datos, mientras que aún les da a sus usuari@s la capacidad de evasión.

📖👐 <strong> Código libre y abierto.</strong>
El Navegador Ceno está potenciado por Ouinet, una biblioteca de código abierto que habilita a desarrollador@s de terceros a incorporar la red Ceno dentro de tus aplicaciones para conectividad punto a punto.

<strong> Por favor ten en cuenta:</strong> Ceno no es una herramienta de anonimización: la información acerca de tu navegación podría ser filtrada a otr@s usuaria@s de Ceno, como así también el hecho de que tu aplicación está proveyendo contenido web particular a otr@s. El contenido accedido con la aplicación puede permanecer almacenado en texto sin cifrar por algún tiempo, de todas formas, puedes purgar todo el contenido almacenado en cualquier momento usando la opción "Purgar Ahora".

<strong>Anticaracterísticas de F-Droid:</strong> El código fuente de Ceno utiliza bibliotecas de Mozilla que no son totalmente libres. Para la versión F-Droid de Navegador de Ceno, parcheamos las partes no libres y construimos estas bibliotecas desde cero.
